## ðŸš« What Kills Your Reach

### Algorithm Penalties
1. **Shadowban** (Reduced visibility)
   - Using banned hashtags
   - Posting inappropriate content
   - Aggressive follow/unfollow
   - Too many hashtags (30 is old advice)
   - Buying followers/engagement

2. **Low-Quality Signals**
   - Stolen/reposted content
   - Low-resolution images
   - Clickbait captions
   - Spam-like behavior
   - External links in every post

3. **Engagement Bait**
   - "Like for X, Comment for Y"
   - Follow/unfollow tactics
   - Comment pods (Instagram detects)
   - Like/comment automation (risky)

### How to Recover from Shadowban
```
1. Stop using banned hashtags (check regularly)
2. Post original, high-quality content only
3. Reduce posting frequency for 1 week
4. Engage genuinely (no automation)
5. Take 48-hour break from Instagram
6. Report problem to Instagram
```
